EMAIL AUTOMATICHE — EMAILJS📧
Setup rapido (5 minuti, gratis):
1. Vai su emailjs.com → crea account gratuito
2. Email Services → Add Service → connetti il tuo Gmail
3. Email Templates → Create Template (usa i parametri sotto)
4. Account → API Keys → copia la Public Key
5. Incolla tutto qui e clicca Salva
L'email a cui arrivano le notifiche di nuove prenotazioni. Corrisponde a {{to_email}} nel template.
Variabili disponibili nel template: {{booking_id}} {{cliente_nome}} {{cliente_email}} {{cliente_tel}} {{oggetto}} {{materiale}} {{quantita}} {{deadline}} {{note}} {{data_invio}}
Usa anche: {{to_name}} {{to_email}} per personalizzare la risposta al cliente
